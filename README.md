# xlsx2pdf 

## Description  
Convert \*.xlsx in the specified directory to PDF.  
**Note:**  
A PDF will be generated with the basename of the target \*.xlsx. Therefore, if a PDF with the same name already exists, it will be overwritten.  

## Usage  
```
$ xlsx2pdf.py
Dir: <Drop folder>
```
